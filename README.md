# TurboType

TurboType is a fast-paced typing game designed to improve your typing speed and accuracy. Test your skills by typing out random words as quickly as possible!

## Gameplay

- Start the game by clicking the "Start" button.
- You have a limited time to type each word correctly.
- Type each word accurately and quickly to score points.
- Your score is based on the number of words you type correctly within the time limit.

## How to Play

1. Click [here](https://evanckennedy.github.io/turbo-type/) to access the game.
2. Once the game loads, click the "Start" button to begin.
3. Type the displayed word as quickly and accurately as possible.
4. Each correctly typed word increases your score.
5. The game ends when the time runs out or all numbers have been typed.

## Features

- User-friendly interface for easy gameplay.
- Random word generation for endless typing challenges.
- Real-time score tracking to monitor your progress.
- Engaging sound effects to enhance the gaming experience.
- Top 9 scores are stored in local storage in the leaderboard.
- Real-time feedback on correctly inputted letters with color changes.
